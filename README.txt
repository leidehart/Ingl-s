LevelUp English — protótipo de app A2→B1/B2

Como usar:
1. Abra o arquivo index.html no Chrome, Edge ou Safari.
2. No celular, envie a pasta/ZIP para o aparelho e abra o index.html pelo navegador.
3. Para fala por microfone, Chrome no Android costuma funcionar melhor.
4. A voz feminina depende das vozes instaladas no seu celular/navegador.
5. O progresso fica salvo apenas no navegador via localStorage.

Limites do protótipo:
- Não usa servidor, login nem API paga.
- A avaliação de pronúncia é aproximada: o navegador transcreve sua fala, e o app avalia a transcrição.
- Para avaliação fonética profissional, seria necessário integrar uma API de speech assessment.
